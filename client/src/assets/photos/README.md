# Presentation Photos

Drop Singapore Botanic Gardens photos in this folder to use them in slide designs.

Recommended names:

- `slide-0.jpg` for the cover slide
- `slide-1.jpg` for the agenda slide
- `slide-2.jpg` for the poll slide
- `slide-3.jpg` for the takeaways slide
- `slide-4.jpg` for the closing slide

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`.

After adding photos, restart the Vite/dev server or rebuild so the new files are bundled.
